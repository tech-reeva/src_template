# CapitalShop
Ecommerce using vanilla js, html, sass and bootstrap

## Features
- Contains Cart page 
- Contains Favourate page
- Login and Sign up
- Log out
- Dark Mode
- Using LocalStoarge
- Product Details Page
- Using Sass and Bootstrap 5
